# DoNotDo

## 食用方法
若想让游戏时可以查看其他人不要做的事情
可以食用
```
scoreboard objectives add broadcast trigger
```
***
开始命令:
```
function donotdo:start
```
## 一些说明
不要做的事情尽可能避免危机情况（比如苦力怕来莉要攻击）
以及一些维持生存基本的事情（比如吃食物）
